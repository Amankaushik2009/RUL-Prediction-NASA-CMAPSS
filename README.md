# Machine Learning-Based Remaining Useful Life Prediction Using NASA C-MAPSS Data

## Project Overview

This repository contains the code developed for my Major Research Project (MRP) in the Master of Science in Data Science and Analytics program at Toronto Metropolitan University.

The objective of this project is to develop and compare machine learning models for predicting the Remaining Useful Life (RUL) of rotating equipment using the NASA C-MAPSS FD001 dataset. The project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model development, model evaluation, and Explainable Artificial Intelligence (SHAP) for model interpretation.

## Repository Contents

- `MRP_CMAPSS_FD001_EDA_Notebook.ipynb` – Data loading, preprocessing, and exploratory data analysis.
- Additional notebooks and source code will be added as the project progresses.

## Dataset

The NASA C-MAPSS FD001 dataset is publicly available from NASA:

https://data.nasa.gov/dataset/cmapss-jet-engine-simulated-data

## Project Workflow

1. Dataset selection
2. Data cleaning and preprocessing
3. Exploratory Data Analysis (EDA)
4. Feature engineering
5. Machine learning model development
6. Model evaluation (MAE, RMSE, and R²)
7. SHAP-based model interpretation
